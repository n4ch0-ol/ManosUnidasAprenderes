ES.E.N.C.I.A

¿Qué es ES.E.N.C.I.A?

ES.E.N.C.I.A es un proyecto que busca reformar ciertos aspectos del sistema de enseñanzas de este colegio para hacerlo más flexible a diversos eventos que podrían realizarse de ser más flexible el sistema

Objetivos del proyecto

Con el proyecto E.S.E.N.C.I.A pretendemos mejorar el sistema para qe sea más flexible y poder realizar, por jemplo, clases alternativas con otra trayectoria. Esto no significa que lo que estamos haciendo esté mal, si no que creemos que el sistema puede mejorarse.
